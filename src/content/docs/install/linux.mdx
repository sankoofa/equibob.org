<Aside type="note">
    While the Flatpak is convenient, it comes with some limitations such as dysfunctional Rich Presence. We recommend
    using native packages for the best experience.
</Aside>

[![Download on Flathub](https://dl.flathub.org/assets/badges/flathub-badge-en.svg)](https://flathub.org/apps/dev.vencord.Vesktop)

<IconHeader tag="h4" iconUrl={SteamIcon.src}>
    SteamOS
</IconHeader>

Open the Discover app, search for Vesktop and install it. To run it in game mode, add it as a non-steam game.

<IconHeader tag="h4" iconUrl={UbuntuIcon.src}>
    deb (Debian / Ubuntu)
</IconHeader>

Download the right deb package and double click it to install.

<LinkButton href="https://vencord.dev/download/vesktop/amd64/deb">x64 Download</LinkButton>
<LinkButton href="https://vencord.dev/download/vesktop/arm64/deb">arm64 Download</LinkButton>

<IconHeader tag="h4" iconUrl={FedoraIcon.src}>
    rpm (Fedora / Red Hat)
</IconHeader>

Download the right rpm package and double click it to install.

<LinkButton href="https://vencord.dev/download/vesktop/amd64/rpm">x64 Download</LinkButton>
<LinkButton href="https://vencord.dev/download/vesktop/arm64/rpm">arm64 Download</LinkButton>

<IconHeader tag="h4" iconUrl={AppImageIcon.src}>
    tarball (Portable)
</IconHeader>

To run the portable version, extract it into a folder and run the `vesktop` binary inside it. Note that all files are needed, not just the binary!
